# Add Two Numbers

A small browser app that adds any two numbers.

## Public app

Open the live app here:

https://ankursinghanianec.github.io/add-two-numbers-app/

## How to use

Open `index.html` in a web browser, enter two numbers, and click **Add Numbers**.

## How to share

- Share the public URL: https://ankursinghanianec.github.io/add-two-numbers-app/
- Send the `index.html` file directly. The other person can double-click it to open the app.
- Send `add-two-numbers-app.zip`. The other person should unzip it, then open `index.html`.

The `http://127.0.0.1:8000/index.html` link only works on the computer where the local server is running.
